# Deploying the Dashboard Live

This project is ready to be hosted as a live dashboard using Streamlit Community Cloud.

## Recommended deployment path

1. Open https://streamlit.io/cloud and sign in with your GitHub account.
2. Click **New app** and choose this repository: `parthnag25/Stock_Foreacsting`.
3. Set the file path to `dashboard.py`.
4. Click **Deploy**.

Once deployed, copy the app URL and add it to your GitHub profile README or this repository README.

## Add a live link to your GitHub profile

In your GitHub profile README repository, include a section like:

```md
## Live Project

- [Live Stock Forecasting Dashboard](https://share.streamlit.io/USERNAME/Stock_Foreacsting/main/dashboard.py)
```

Replace `USERNAME` with your GitHub username and the actual Streamlit app URL after deployment.
